Git

  • git ls- falis - расшифровывается как "List files in the directory", что переводится как "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • git checkout - переключение между ветками.
  • git status-Перечисляет все новые или изменённые файлы, которые нуждаются в фиксации
  • git reset [коммит]- Отменяет все коммиты после заданного, оставляя все изменения в рабочей директории
  • git mv - расфшифровывается как"Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить, переименовать или удалить файл.
