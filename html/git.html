Git

  • git init — инициализирует новый репозиторий Git в текущем каталоге.
  • git add — добавляет указанный файл в индекс.
  • git commit -m "message" — создаёт новую фиксацию с указанным сообщением.
  • git status — показывает текущее состояние репозитория.
  • git branch — показывает список веток.
  • git checkout branch — переключается на указанную ветку.
  • git reset commit — сбрасывает текущую ветку до указанного коммита.
  • git push remote branch — отправляет изменения в указанную ветку на удалённый сервер.
